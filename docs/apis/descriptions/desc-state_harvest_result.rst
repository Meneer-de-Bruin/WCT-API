| **state**
| The state of a harvest result is an integer with the following values:

========= ===============
**State** **Description**
--------- ---------------
  0       
  1       Endorsed
  2       Rejected
  3       Indexing
  4       Aborted
========= ===============

The list of possible state values for harvest results can be retrieved with API :doc:`api-harvest_result_states_GET`.
